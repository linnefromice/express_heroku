# express_heroku

## Technology

- express ... node.js web application framework
- ejs ... template engine with express
- jest ... test tool for javascript
- heroku ... PaaS

## Command

### Local

- git push origin master
- npm start

### Heroku

- git push heroku master
- heroku open
